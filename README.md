# eslint-config-onur

## Install 

### npm
```sh
$ npm install --save-dev eslint prettier eslint-config-onur
```

### yarn 
```sh
$ yarn add --dev eslint prettier eslint-config-onur
```

## Configuration

Add the config to your `.eslintrc.js` file

```js
module.exports = {
  extends: 'eslint-config-onur'
}
